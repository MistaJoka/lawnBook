# Example: Solo Landscaper App Context

## One-Sentence App

A mobile-first offline-capable app for a solo landscaper to manage clients, jobs, estimates, photos, and daily work status from a phone.

## Target User

A non-technical solo landscaper using a Samsung Galaxy phone while driving between jobs and working in areas with weak signal.

## MVP Features

- Client list
- Job list
- Job status board
- Estimate notes
- Before/after photos
- Offline draft saving
- Simple daily schedule

## Phase 1 Done Means

The user can open the app, create a client, create a job, update job status, attach notes, and see the work later after refresh.
