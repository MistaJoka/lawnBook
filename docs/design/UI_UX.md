## Screen Questions

- What should the first screen show?
- What is the most important button?
- What information should be visible without scrolling?
- What can be hidden behind details?
- What should be reachable in one tap?

## Mobile First Rules

- Large tap targets.
- Clear primary action.
- Minimal typing.
- Fast status updates.
- Visible offline/sync state if needed.

## Beginner Questions

- What colors should be used?
- What layout should be used?
- What should buttons look like?
- What should empty states say?
- What should loading states show?

## Advanced Questions

- What interactions reduce cognitive load?
- What can be done one-handed?
- What needs confirmation before saving or deleting?
- What actions should be undoable?
- What accessibility needs matter?
